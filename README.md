# WebSocket Applications
## FastAPI
To run a WebSocket application on FastAPI:
```
cd python-fastapi-app
pipenv shell
uvicorn main:app --reload
CTRL+C
exit
cd ..
```
## SANIC
To run a WebSocket application on Sanic:
```
cd python-sanic-app
pipenv shell
sanic main
CTRL+C
exit
cd ..
```
